구현 기능

할인 가격 계산 기능

버튼을 클릭하면 금액과 할인율로 계산해서 할인된 가격을 보여줍니다.

가격 입력 기능

금액(정가)을 입력할 수 있습니다.

할인율 입력 기능

할인율을 입력할 수 있습니다.

배경과 목적
요즘 사람들은 모바일 앱을 많이 사용합니다. 따라서 하이브리드 앱 개발자가 될 것에 대비해서 기초 소양를 키우기 위해 저만의 앱을 구현하였습니다.
배운 점
  • React Native의 CSS로 레이아웃을 중앙에 배치하는 법을 알게 되었습니다.
  • 버튼을 터치했을 때 이벤트를 호출하는 법을 알게 되었습니다.
  • 레이아웃의 크기를 조절하고 버튼의 색깔을 변경하는 등 UI를 꾸미는 법을 알게 되었습니다.
진행 절차
  1. ’React Native Tutorial’ 강의를 15강까지 들으면서 하나씩 따라하였습니다.
  2. 입력칸 내의 글자색을 변경하고 버튼의 색깔을 변경하는 등 UI를 꾸몄습니다. 그리고 앱 이름과 아이콘을 변경하였습니다.
  3. 버튼을 클릭하면 할인된 가격이 보이도록 하였습니다.
  4. React Native의 CSS로 레이아웃을 중앙에 배치하였습니다.
  5. 이렇게 해서 원하는 프로그램을 구현할 수 있었습니다.
