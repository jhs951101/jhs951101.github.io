ボタンをクリックしたら、金額と割引率で計算して割引価格を見せます。

背景と目的
最近の人々は、モバイルアプリの多くを使用しています。 そのため、ハイブリッドアプリの開発者になるために基礎を養うために独自のアプリを実装しました。
学んだこと
  • React NativeのCSSでレイアウトを中央に配置する方法を学びました。
  • ボタンをタッチしたときにイベントを呼び出す方法を学びました。
  • レイアウトのサイズを変更し、ボタンの色を変更するなど、UIをどのように飾るかを学びました。
進行手順
  1. ’React Native Tutorial’ 講義を15講義まで聞きながら一つずつフォローしました。
  2. 入力欄の文字色を変更したり、ボタンの色を変更したりするなど、UIを飾りました。 そして、アプリの名前とアイコンを変更しました。
  3. ボタンをクリックしたら割引価格が表示されます。
  4. React NativeのCSSでレイアウトを中央に配置しました。
  5. このようにして目的のプログラムを実装できました。
