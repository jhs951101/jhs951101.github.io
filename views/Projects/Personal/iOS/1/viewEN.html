Background and Purpose
Nowadays, people use mobile apps a lot. Therefore, I implemented my own app to develop basic knowledge in preparation for becoming an iOS app developer.
What I Learned
  • I learned how to center a layout with constraints and how to place objects at the bottom.
  • I learned how to use the Stack View
  • I learned how to call an event when a button is touched.
  • I learned how to use objects in the layout, such as input fields and labels, in Swift.
  • I learned how to decorate the UI, such as resizing the layout and changing the color of the buttons.
Process
  1. I followed the ‘iOS Tutorial (2020) How To Make Your First App’ lecture one by one.
  2. I realized that the Vertical Stack View cannot be arbitrarily adjusted horizontally while implementing my own program. So I centered the View with constraints and adjusted the horizontal length.
  3. I renamed objects in the layout and reconnected to Swift. At this time, I pressed the Command key and dragged with the mouse to connect. And when you click the button, you can see the discounted price.
  4. I decorated the UI by changing the color of the text in the input field and changing the color of the button. And changed the app name and icon.
  5. In this way, I was able to implement the program I wanted.
