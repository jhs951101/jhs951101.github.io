Implemented Functions

A function to Attack Enemies

The turret can attack enemies within range. This way you can subdue enemies before they reach their destination.

A function to Create Enemies and Move them

When the battle time comes, an enemy is created and then moves to the destination. Life is consumed when the enemy reaches the destination.

A function to Buy Turrets and Place them

Turrets can be placed by purchasing new ones. And you can enhance the attack power of the turret.

Background and Purpose
People use smartphones a lot these days, so I made my own game to make creative smartphone games to provide services to people, and to prepare for becoming a game developer.
What I Learned
  • I learned how to handle Collision in Unity and the difference between Collision and Trigger.
  • I learned how to start and stop periodic processing with Coroutine and IEnumerator in Unity.
Process
  1. I created a Unity 3D project and implemented the land and roads.
  2. Enemies move along the path when they appears. I implemented this by placing a Collider in the middle of the road and changing the destination whenever a collision occurs.
  3. I implemented the UI to create a new turret and upgrade attack power.
  4. I had a new turret created when you touch the location to create it. Similarly, I had the turret moved when you touch the turret and touch a new position to move it.
  5. I had turrets attack enemies if they comes within range of the turret.
  6. I had enemies appear after leveling up every 60 seconds.
  7. In this way, I was able to implement the program I wanted.
