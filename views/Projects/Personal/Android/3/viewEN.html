Background and Purpose
Nowadays, people use mobile apps a lot. Therefore, in preparation for becoming an Android app developer, I implemented my own app to develop basic knowledge.
What I Learned
  • I learned how to run it after building a server on AWS and Cafe24.
  • I learned how to send HTTP Responses in PHP depending on the situation.
  • I learned how to build an Android server with PHP and how to connect and build a database with MariaDB.
  • I learned how to send HTTP Request from Android to Volley, and the difference between GET and POST.
  • I learned how to implement a client in Android.
Process
  1. I implemented the client in Android. At this time, HTTP Request was sent to Volley.
  2. I built a server with PHP, built a database with MariaDB, and then linked.
  3. I launched a server on AWS and Cafe24. However, when it was launched on AWS, an error occurred that could not even access the database. So, after creating a new MariaDB user, I gave this user all privileges and used it on the server. This is how I made it possible to access the database.
  4. In this way, I was able to implement the program I wanted.
