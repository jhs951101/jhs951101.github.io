Implemented Functions

A function for Zombies to Be Falled

Numerous zombies fall from top to bottom.

A function to Remove Zombies

You can remove falling zombies with a touch. Life is consumed when zombies reach the bottom.

A function to Touch Circles

Touch the given circles within the time limit to get rid of them all. If you can't get rid of it, life is consumed.

  2. The clicker game took longer than I thought because it was the first time I made it. The process of making the game itself was the most difficult. So there were a lot of errors, such as not responding when touched and zombies not dying.
  3. I modified source codes and then executed them continuously in order to solve these errors.
  4. If this doesn't work, I searched for an error message on Google, researched a solution to the problem, and applied it to my program.
  5. However, when I fixed one error, another error popped up, and I did a lot of searching on Google to solve it.
  6. In this way, I was able to implement the program I wanted.
