Implemented Functions

A function to Turn On Light on LED

The LEDs near the stage and the LEDs hanging above it twinkle.

A function to Detect Brightness

Light or dark determines how the LED flashes. When it's dark, the LED flashes a lot.

A function to Run Motors

When it is dark, the LED will flash and the motor will run, making it a moving LED.

Background and Purpose
There was a task to make a work with Arduino in an introductory computer system class. It was created to make the most creative work possible.
What I Learned
  • I learned how to design and implement hardware with Arduino
  • I learned how to use LEDs, a brightness sensor, and a servo motor.
  • I learned how to implement the polling method with Arduino and the difference between the polling method and the interrupt method.
Process
  1. Before programming the Arduino, I implemented the stage model using boxes, tapes, and wooden chopsticks.
  2. I found each example program for LED, brightness sensor, and servo motor in the lecture material, and modified it to my liking and merged it into one.
  3. I used a polling method so that it behaves differently depending on whether the light is dark or bright.
  4. There was some trial and error, but I checked the code closely and corrected it.
  5. In this way, I was able to implement the program I wanted.
