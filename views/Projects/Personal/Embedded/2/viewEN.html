Implemented Functions

A function to Move Characters

You can move your character with the joystick.

A function to Shoot objects with Laser Gun

You can fire the laser gun by pressing the button.

A function to Be Catched by Webs

If it touches the web, the character will be unable to do anything. If both characters get caught in the web, they lose.

A function to Win a game

If both characters eat hearts at the same time, you win.
