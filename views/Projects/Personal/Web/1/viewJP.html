実装機能

ソニックのキャラクターについて見る機能

ソニックキャラクターに関する情報を見ることができます。 そしてソニックゲームに関する情報も見ることができます。

ログインと会員登録する機能

ホームページにログインできます。 まだ会員でない場合は、会員登録できます。

1:1のお問い合わせや提案の機能

管理者に個人的なお問い合わせまたは提案も可能です。 この内容は管理者の電子メールで送信されます。

背景と目的
最近、企業ごとに存在するのがまさにホームページです。 このようにWeb開発者になることに備えて基礎を固めるために私だけのホームページを実装しました。
学んだこと
  • ウェブページでHTTPリクエストを送信する方法と、GETとPOSTの違いを学びました。
  • JSP、CSS、JavaScriptでWebページを実装する方法を学びました。
  • JavaからTransportにSMTPを使用してメールを送信する方法と、SMTP、POP3、IMAPの違いを学びました。
  • JavaとServletでWebサーバーを構築する方法とMySQLでデータベースを構築してJDBCと連携する方法を学びました。
進行手順
  1. 既存のホームページに不要な機能が思った以上に削除されました。
  2. JavaとServletでWebサーバーを構築し、MySQLとJDBCでデータベースを構築した後に連携しました。
  3. 会員登録、ログイン、1:1問の時、ウェブサーバでsubmitになるように修正し、サンプルコードを見ながら実装しました。
  4. 全体的にうまく実装しましたが、ソニックゲームを見るときに10個ずつ出るようにしましたが、エラーメッセージが表示されました。
  5. 実装したコードを綿密に確認し、悩んだ最後に、10個ずつ出るように実装できました。
  6. 1：1の場合、WebサーバーでTransportとSMTPを使用して運営者のEメールで内容を送信するようになりました。
  7. このようにして目的のプログラムを実装できました。
能力強化のために最も努力したこと
JSP&Servletでホームページ実装、Webサーバー構築、データベース(MySQL)と連動
全体構造
使用技術
JSP, CSS, JavaScript, Java, Servlet, HTTP, MySQL
開発環境
Eclipse IDE, PHPMyAdmin
