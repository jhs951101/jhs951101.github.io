Implemented Functions

A function to Shoot objects with Gun

You can click on an approaching zombie to shoot a pistol to remove it. This will keep the zombies from coming to the bottom.

A function to Use a Skill

When the skill gauge is full after a certain period of time, you can use skills to subdue all the zombies.

A function for Zombies to Come

Zombies are coming down. If you reach the bottom, life is consumed.

Background and Purpose
There was once an intra-school Scratch competition in high school. In order to provide services to people by making creative games, and to prepare for becoming a game developer, I created and submitted my own game.
What I Learned
  • I learned how to implement a game with Scratch.
Process
  1. I borrowed a Scratch book from the library and selected and executed only the necessary examples.
  2. Referring to the examples in the Scratch book, I implemented necessary things little by little, such as clicking and suppressing zombies, decreasing life when zombies approach a wall, and raising the skill gauge little by little. Actually, it was my first time making a game with pure programming, so I was very worried.
  3. I modified the code after combining the implementations so far.
  4. In this way, I was able to implement the program I wanted.
