확진자 발생 시 알림 기능

김포시 내에 확진자가 발생하면 수시로 알립니다.

기록 남기기 기능

확진자 발생에 대한 정보를 기록으로 남길 수 있습니다.

사회에 기여하려고 한 점
코로나19 확진자 발생 시 알림 없이 기록으로만 남길 수 있으므로 알림에 방해받지 않고 확진자 발생 정보를 얻을 수 있습니다.
배경과 목적
기존의 코로나19 긴급재난메시지는 확진자 발생 시 바로 전송되므로 시도 때도 없이 알림이 울려서 방해된다는 문제가 있습니다. 그렇다고 긴급재난메시지를 차단하면 확진자 발생 정보를 기록으로 남길 수 없다는 문제가 생깁니다. 그래서 확진자 발생 정보를 알림 없이도 얻을 수 있도록 챗봇을 구현하였습니다.
배운 점
  • Python 프로그램으로 Telegram 챗봇을 컨트롤하는 법을 알게 되었습니다.
  • Python에서 BeautifulSoup로 크롤링 하는 법을 알게 되었습니다.
  • Python에서 split() 함수로 문자열을 분리하는 법을 알게 되었습니다.
진행 절차
  1. ’챗봇으로 나만의 랜선여친 만들기’ 소스 코드를 다운받았습니다. 여기서, 소스 코드는 'telegram.ext'라는 라이브러리를 사용하였습니다.
  2. 웹 크롤링 예제를 구글에서 찾아 실행하였습니다. 여기서, 예제는 BeautifulSoup를 사용하였습니다.
  3. 김포시청 사이트에서 확진자 발생 글만 나오게 한 다음 ‘소스 보기’를 클릭해서 웹페이지 코드를 면밀히 분석하였습니다. 이 후 split() 함수를 활용해서 게시글의 제목, 내용, 링크만 가져오도록 하였습니다.
  4. 유튜브에 나와있는 대로 챗봇을 새로 만들었고, 웹 크롤링 예제와 합쳐서 크롤링을 한 후 통보하도록 하였습니다.
  5. 확진자 발생 게시글이 새로 올라왔는지 알아야 하는데, 이것은 텍스트 파일에 최근 게시글의 제목을 저장해서 주기적으로 비교하는 방식을 사용하였습니다. 만약 텍스트 파일이 없다면 새로 생성하였습니다.
  6. 이렇게 해서 원하는 프로그램을 구현할 수 있었습니다.
