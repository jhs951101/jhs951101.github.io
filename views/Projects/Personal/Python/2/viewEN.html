Things to Contribute to Society
When a confirmed case of COVID-19 occurs, it can only be recorded as a record without notification, so you can get information on the occurrence of confirmed cases without being disturbed by notifications.
Background and Purpose
The existing COVID-19 emergency message is sent immediately when a confirmed person occurs, so there is a problem in that it is interrupted by a notification sounding at any time. However, if the emergency message is blocked, there is a problem that the information about the number of confirmed cases cannot be recorded. So, we implemented a chatbot so that you can get information about the number of confirmed cases without notification.

Process
  1. I downloaded the source code for ‘Make your own LAN girlfriend with chatbot’. Here, the source code used a library called 'telegram.ext'.
  2. I found an example of web crawling on Google and executed it. Here, the example uses BeautifulSoup.
  3. On the website of Gimpo City Hall, only the articles of confirmed cases appeared, and then clicked 'View Source' to closely analyze the web page code. After that, I used the split() function to get only the title, content, and link of the post.
  4. I created a new chatbot as shown on YouTube, and combined it with the web crawling example to notify after crawling.
  5. I need to know if there are new posts with confirmed cases. This is done by storing the titles of recent posts in a text file and comparing them periodically. A new one is created if the text file does not exist.
  6. In this way, I was able to implement the program I wanted.
