実装機能

パズルのピースを変える機能

パズルのピースを2つクリックして、反対の方法でゲームを進めます。

完成したイメージを見せる機能

難しすぎる場合は、完成画像を見てゲームを進めることができます。

パズルの完成をチェックする機能

パズルが完成すると、「お疲れ様です！」という窓が浮かびます。

イメージを選択する機能

パズルゲームを進める画像を選択できます。 そして、自分のイメージを呼んでゲームを進めることもできます。

社会に貢献しようとしたこと
クリエイティブなゲームを制作してリリースするので、ゲームをしながら一味違う楽しさを感じることができます。
背景と目的
創造的なゲームを作って人々にサービスを提供するために、ゲーム開発者になることに備えて私だけのゲームを制作することになりました。
学んだこと
  • Java Swingでコンピュータプログラムを実装する方法を学びました。
進行手順
  1. 2年生 2学期専攻科目「オブジェクト指向プログラミング」の時に画像を複数のボタンで切り取るプログラムを提供していただきました。 これを2次元配列にボタンをランダムに配置した後、ボタン2つをクリックすれば互いに入れ替えるように修正しました。
  2. 上部メニューバーのヒントボタンを追加し、クリックすると元の画像が表示されます。
  3. ゲームを開始する前に、複数の写真のいずれかを選択する機能と、目的の写真を読み込む機能を実装しました。
  4. 希望の写真を読み込むときにサイズが大きすぎる写真は、読み込むとエラーになります。 そのため、画像を読み込むときにサイズを合わせる機能を追加しました。
  5. 最後にゲームスタート画面を実装しました。
  6. このようにして目的のプログラムを実装できました。
能力強化のために最も努力したこと
Java Swingで実装された既存のプログラム分析と創造的なゲームで再構築
