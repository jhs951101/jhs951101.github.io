What I Learned

Process
  1. A professor provided a program for cutting images with multiple buttons during the ‘Object-Oriented Programming’ class in the 2nd semester of your sophomore year. After arbitrarily arranging buttons in a two-dimensional array, we have modified them so that they are swapped when clicking two buttons.
  2. I added a hint button on the top menu bar to show the original image when clicked.
  3. I implemented a function to select one of several pictures before starting the game and a function to load the desired picture.
  4. When loading a desired picture, an error occurs when loading a picture that is too large. So, I added a function to adjust the size when loading an image.
  5. Finally, I implemented the game start screen.
  6. In this way, I was able to implement the program I wanted.
