Implemented Functions

A function to Move and Jump

You can move by pressing the arrow keys, and you can also jump by pressing the space bar.

A function to Load Information about a Plant

If you click a button after approaching a plant within the metaverse environment, you can view various information including the actual appearance of the plant.

A function to Access Shop

You can access the farmer's shop to view and purchase crops currently on sale.

A function to Purchase Plants

You can purchase crops with the cash you have. If you run out of cash, you can recharge it.

Things to Contribute to Society
You can also grow crops in the metaverse environment, so you don't have to go to the place where you grow your crops. It is also possible for the computer to manage the crops by itself through an artificial intelligence program.
Background and Purpose
The metaverse environment has not yet been established in Korea much. Most representatively, there is currently no service to grow crops in the metaverse environment in Korea. So, we have implemented a program that can grow crops in the metaverse environment.

Process

  2. I built a server with PUN2 and built a database with Firebase.
  3. I implemented a payment function with the NH open platform and had the weather data loaded from a public data portal, but this was the most difficult.
  4. So, I applied the example of sending HTTP Request from Google to my program and linked Unity with the NH Open Platform.
  5. And, I modified the source code to load weather data from a public data portal after rewatching and following the lecture given by the hosting company.
  6. I carefully observed contents which designers designed in Zeplin and implemented it directly in Unity.
  7. In this way, I was able to implement the program I wanted.
  8. I uploaded a demonstration video to Google Drive after filming it, and sent it to the planner.
